# 문제 설명

정수 배열 `arr`가 주어집니다. `arr`의 각 원소에 대해 다음 작업을 반복합니다:

- 값이 50보다 크거나 같은 짝수라면 2로 나눕니다.
- 값이 50보다 작은 홀수라면 2를 곱하고 다시 1을 더합니다.

이러한 작업을 **x번 반복한 결과**인 배열을 `arr(x)`라고 표현했을 때,  
`arr(x) = arr(x + 1)`인 **x**가 항상 존재합니다.  
이러한 **x** 중 가장 작은 값을 반환하는 `solution` 함수를 완성해 주세요.

## 문제 조건

- **두 배열이 같음**은:
    1. 배열의 길이가 서로 같으며,
    2. 같은 인덱스의 원소가 각각 동일함을 의미합니다.

## 제한 사항

- `1 ≤ arr의 길이 ≤ 1,000,000`
- `1 ≤ arr의 원소의 값 ≤ 100`

---

## 입출력 예

| arr                      | result |
|--------------------------|--------|
| `[1, 2, 3, 100, 99, 98]` | `5`    |

---

## 입출력 예 설명

### 예제 #1

`arr`가 반복 작업에 따라 변하는 모습은 아래와 같습니다:

| 반복 횟수 | arr                       |
|-------|---------------------------|
| 0     | `[1, 2, 3, 100, 99, 98]`  |
| 1     | `[3, 2, 7, 50, 99, 49]`   |
| 2     | `[7, 2, 15, 25, 99, 99]`  |
| 3     | `[15, 2, 31, 51, 99, 99]` |
| 4     | `[31, 2, 63, 51, 99, 99]` |
| 5     | `[63, 2, 63, 51, 99, 99]` |
| 6     | `[63, 2, 63, 51, 99, 99]` |

이후로 `arr`는 변하지 않으며,  
`arr(5) = arr(6)`이므로 **5**를 반환합니다.
