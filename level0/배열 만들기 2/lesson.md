## 핵심 아이디어

```python
n = 5 * int(bin(i)[2:])
```

이 한 줄이 핵심인데, 순서를 나눠서 보면:

### 1. bin(i)
- 정수 `i`를 이진수 문자열로 변환
- 예:
  - `bin(1)` → `'0b1'`
  - `bin(2)` → `'0b10'`
  - `bin(3)` → `'0b11'`

### 2. [2:]
- `'0b'` 제거
  - `'1'`, `'10'`, `'11'`

### 3. int(...)
- 이걸 10진수로 해석
  - `int('1')` → `1`
  - `int('10')` → `10`
  - `int('11')` → `11`

### 4. * 5
- 각 자릿수에 5를 곱한 효과
  - `1 * 5 = 5`
  - `10 * 5 = 50`
  - `11 * 5 = 55`

**결과적으로:** 0과 1로 이루어진 이진수를 → 0과 5로 이루어진 숫자로 변환

## 생성되는 숫자 패턴

`i`가 증가하면서 만들어지는 `n` 값:

| i | bin(i) | 변환 | n  |
|---|--------|------|----|
| 1 | 1      | 1    | 5  |
| 2 | 10     | 10   | 50 |
| 3 | 11     | 11   | 55 |
| 4 | 100    | 100  | 500|
| 5 | 101    | 101  | 505|

즉, `5, 50, 55, 500, 505, 550, 555 …` → 0과 5로만 이루어진 숫자들

## 반복문

```python
while True:
    if n > r: break
```

- `r`보다 커지면 종료

## 조건 필터링

```python
if l <= n <= r:
    answer.append(n)
```

- 범위 안에 있으면 리스트에 추가

## 결과 반환

```python
return [-1] if len(answer) == 0 else answer
```
